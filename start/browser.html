WuX-UI Browser Compatibility.

Fully guaranteed compatibility

WuX-UI with the recommended Polyfill is fully compatible with at least two versions of the major browsers (Chrome, Firefox, Edge, Safari).

Internet Explorer, NetSurf and other obsolete browsers have been tested, so please find another framework if you have such needs.

As for the non-mainstream browsers such as Opera, Brave, Vivadi, etc., they are untested and should work fine without any surprises, so please raise an issue on Github if there are any bugs, we would be very grateful.
