WuX-UI Quick-Start

Quickly produce projects with WuX-UI.

Usage

WuX-UI requires only a simple template file for rapid development without a tedious build process.

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>WuX-UI Demo</title>
    <link href="https://cdn.jsdelivr.net/gh/wux-ui/wux-ui/dist/wux.css">
    <!--Polyfill-->
    <link href="https://cdn.jsdelivr.net/gh/wux-ui/wux-ui/dist/polyfill/attr-polyfill/attr-polyfill.js">
    <link href="https://cdn.jsdelivr.net/gh/wux-ui/wux-ui/dist/polyfill/dialog-polyfill/dialog-polyfill.js">
  </head>
  <body>
    <h1>Hello, World.!</h1>
  </body>
</html>
