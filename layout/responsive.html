WuX-UI Responsive

The foundation for building responsive web pages, including grids, etc.

Grid

The responsive grid system neatly arranges the content of your web pages while maintaining the ability to display perfectly on multiple devices.

WuX-UI has 5 built-in sizes, xs / sm / md / lg / xl, which can be used for row or col.

Row

Use wux-row-[1~12] to divide each Col in Row equally.

Col

Use wux-col-[1~12] to customize the size share of each Col in the Row.
