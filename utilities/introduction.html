WuX-UI Utilities Introduction

Intro

WuX-UI has a very powerful Utilities System Alpha. With it, you can easily define CSS values in HTML directly.

Although the system is still unstable and requires some functionality through Polyfill. However, this feature was defined 10 years ago in CSS-Values-3 20110906 (It has recently been postponed to CSS-Values-5 and does not appear after CSS-Values-4 20211016, which is sad.), and it is believed that browser vendors will soon support this feature.

Polyfill

Polyfill must be used to support WuX-UI Utiltiles System Alpha before the browser supports the attr() function.

We recommend using attr-polyfill provided by ZhangXinXu.

Will support in the future

Future Utilities will support settings in the Before and After Pseudo-element , and support more CSS attributes.
