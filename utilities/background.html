WuX-UI Background

Background Color

Use the bg-color attribute to apply the color.

Example

This sentence has background color.

<p bg-color="#ABCDEF">This sentence has background color.</p>

Background Image

Use the bg-img attribute to apply a background image (URL).

Example

This sentence has background image.

<p bg-img="https://github.githubassets.com/pinned-octocat.svg">This sentence has background image.</p>

Background Repeat

The bg-repeat attribute can set whether the picture should be repeated or not, see MDN for details.

Example

This sentence has background image and no-repeat.

<p bg-img="https://github.githubassets.com/pinned-octocat.svg" bg-repeat="no-repeat">This sentence has background image and no-repeat.</p>

Background Position

The bg-position attribute positions the background image, see MDN for details.

You can also use bg-positon-x and bg-position-y to set the position of the background image for the horizontal and vertical axes respectively.

Example

This sentence has background image, no-repeat and on the right.

<p bg-img="https://github.githubassets.com/pinned-octocat.svg" bg-repeat="no-repeat" bg-position="right">This sentence has background image, no-repeat and on the right.</p>
