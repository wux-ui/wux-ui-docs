WuX-UI Borders Can use direction.

Border Color

Use the border-color attribute to apply the color.

Border Width

Use the border-width attribute to apply the width.

Border Style

Use the border-style attribute to apply the style, see MDN for details.

Border Color/Width/Style Example

This is an example.

<p border-width="2px" border-color="#ABCDEF" border-style="dotted">This is an example.</p>

Border Radius

Use the border-radius attribute to apply the border radius.

Example

This is an example.

<p border-width="2px" border-color="#ABCDEF" border-style="dotted" border-radius="6px">This is an example.</p>
