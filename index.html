Components

Alert
The Alert component feedback to users on various messages using simple component prompts.
Go
Badge
The Badge component is a simple and clear numeric prompt in the top right corner of the component.
Go
Blankslate
The Blankslate component provides error messages when the original information is missing.
Go
Breadcrumb
The Breadcrumb component is used to express the site hierarchy, all items must be used as links.
Go
Button
The Button component is used to guide the user through the most basic actions.
Go
Card
The Card component is used as a basic container and is available in both flat and floating styles.
Go
Collapse
The Collapse component is used for scenes where information needs to be collapsed.
Go
Dialog
The Dialog component can be used as an important reminder to the user.
Go
Dropdown
The Dropdown component is used to hold less important content in a limited space.
Go
Form-Checks
The Form-Checks component is used when multiple options can be selected in the selection.
Go
Form-Input
The Form-Input component is an input box, which can be divided into single line and multi-line.
Go
Form-Radios
The Form-Radios component is used when only a single option can be selected in the selection.
Go
Form-Range
The Form-Range component is used when fully accurate values aren't need.
Go
Form-Select
The Form-Select component is single-selectable only and takes up only a line of space.
Go
Form-Upload
The Form-Upload component is used in forms where files need to be uploaded.
Go
Header
The Header component is placed at the top of the page and is used to navigate the page.
Go
Jumbotron
The Jumbotron component is used to indicate the page focus title, or tagline.
Go
List
The List component is used to display multiple contents and can also be used as navigation.
Go
Loading
The Loading component indicates that the content or component is loading.
Go
Progress
The Progress component acts as a progress bar and can be used together with Loading.
Go
Search
The Search component is more suitable for search purposes than the Input component.
Go
Tab
The Tab component is used to distinguish between multiple tabs and select them.
Go
Tables
The Tables component is used to display multiple profiles in a neat and easy-to-read manner.
Go
Tag
The Tag components can represent topics and display them in a small space.
Go
Tooltip
The Tooltip component is displayed on mouse hover and is used to prompt the user.
Go

Container
