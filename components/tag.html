WuX-UI Tag

The Tag components can represent topics and display them in a small space.

Example

Tag 
<span class="wux-tag">For</span>

Colors

The default Tag color is Primary, and you can change the Tag color using wux-tag-[color], see Customize/Colors for details.

Customize

SCSS Variables

/*Default*/
$tag-border-radius: 4px;
$tag-margin: 2px;
$tag-padding: 6px 8px;

$tag-background-color-degree: 200;

/*
 * Color-Degree is used to obtain the colors defined in customize/color/colors using Map. It is not recommended to modify Map directly to replace saturation.
 */

/*Dark*/
$tag-background-color-degree-dark: 300;

CSS Variables

:root {
  --tag-border-radius
  --tag-margin
  --tag-padding

  --tag-background-color-primary
  --tag-background-color-secondary
  --tag-background-color-success
  --tag-background-color-info
  --tag-background-color-warning
  --tag-background-color-error
  --tag-background-color-light

  --tag-text-color-primary
  --tag-text-color-secondary
  --tag-text-color-success
  --tag-text-color-info
  --tag-text-color-warning
  --tag-text-color-error
  --tag-text-color-light
}
