WuX-UI Alert

The Alert component feedback to users on various messages using simple component prompts.

Example

A primary alert. 
<div class="wux-alert wux-alert-primary">A alert</div>

Colors

The default Alert color is Primary, and you can change the Alert color using wux-alert-[color], see Customize/Colors for details.

Secondary
Success

Icons

The icon library is not built into WuX-UI, we recommend using the Font-Awesome icon library.

To use icons in Alert, you just need to use the wux-alert-icon class, which makes the icons unselected, and you can also use Emoji.

:D Kawaii.

Options

You can apply the wux-alert-option-group class to Alert to display the close button, etc. The default orientation of the Option-Group is on the right.

SCSS Variables

/*Default*/
// Layout
$alert-border-radius
$alert-border-width
$alert-font-size
$alert-padding
$alert-margin-bottom

// Color-Degree
$alert-background-degree
$alert-border-degree
$alert-icon-degree

/*
 * Color-Degree is used to obtain the colors defined in customize/color/colors using Map. It is not recommended to modify Map directly to replace saturation.
 */

/*Dark*/
$alert-background-degree-dark
$alert-border-degree-dark
$alert-icon-degree-dark
