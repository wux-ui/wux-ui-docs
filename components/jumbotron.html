WuX-UI Jumbotron

The Jumbotron component is used to indicate the page focus title, or tagline.

Example

WuX-UI is a CSS Framework,
but more.

WuX-UI is a Modern CSS Framework.

Offical Website Github 
<div class="wux-jumbotron">
  <h1 class="wux-jumbotron-title">
    <b class="wux-jumbotron-title-bold">WuX-UI</b> is a CSS Framework, <br>but more.
  </h1>
  <p class="wux-jumbotron-subtitle"> WuX-UI is a Modern CSS Framework. </p>
  <div class="wux-jumbotron-btn-group">
    <a href="https://wux-ui.tk" class="wux-btn wux-btn-round">Offical Website</a>
    <a href="https://github.com/wux-ui/wux-ui" class="wux-btn wux-btn-secondary wux-btn-text">Github</a>
  </div>
</div>

Align

The component can use the align property to define the orientation of the content.

Customize

SCSS Variables

/*Default*/
$jumbotron-background
$jumbotron-padding
$jumbotron-text-align

$jumbotron-title-font-size
$jumbotron-title-margin-y
$jumbotron-title-text-color

$jumbotron-title-bold-color

$jumbotron-subtitle-font-size
$jumbotron-subtitle-margin-y

$jumbotron-button-group-margin-y

/*Dark*/
$jumbotron-background-dark

$jumbotron-title-bold-color-dark
$jumbotron-title-text-color-dark

CSS Variables

:root {
  --jumbotron-background
  --jumbotron-padding
  --jumbotron-text-align

  --jumbotron-title-font-size
  --jumbotron-title-margin-y
  --jumbotron-title-text-color
  
  --jumbotron-title-bold-color
  
  --jumbotron-subtitle-font-size
  --jumbotron-subtitle-margin-y
  
  --jumbotron-button-group-margin-y
}
