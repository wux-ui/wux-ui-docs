WuX-UI Progress

The Progress component acts as a progress bar and can be used together with Loading.

Example

<progress class="wux-progress" value="80" max="100"></progress>

Customize

SCSS Variables

/*Default*/
$progress-background-color
$progress-border-width
$progress-border-color
$progress-border-radius
$progress-height
$progress-line-color

/*Dark*/
$progress-background-color-dark
$progress-border-color-dark
$progress-line-color-dark

CSS Variables

:root {
  --progress-background-color
  --progress-border-width
  --progress-border-color
  --progress-border-radius
  --progress-height
  --progress-line-color
}
