WuX-UI Tab

The Tab component is used to distinguish between multiple tabs and select them.

Example

Test For #1
Test For #2
Test For #3 
<div class="wux-tab">
  <input class="wux-tab-item" type="radio" name="tab-ex" id="tab-1" checked />
  <label class="wux-tab-item" for="tab-1">1</label>
  <div class="wux-tab-content">
    Test For #1
  </div>
  <input class="wux-tab-item" type="radio" name="tab-ex" id="tab-2"/>
  <label class="wux-tab-item" for="tab-2">2</label>
  <div class="wux-tab-content">
    Test For #2
  </div>
  <input class="wux-tab-item" type="radio" name="tab-ex" id="tab-3"/>
  <label class="wux-tab-item" for="tab-3">3</label>
  <div class="wux-tab-content">
    Test For #3
  </div>
</div>
