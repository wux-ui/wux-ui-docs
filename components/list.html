WuX-UI List

The List component is used to display multiple contents and can also be used as navigation.

Example

  • Hello
  • World
  • Focus
<ul class="wux-list-group">
  <li class="wux-list-item wux-list-item-action">Hello</li>
  <li class="wux-list-item wux-list-item-action">World</li>
  <li class="wux-list-item wux-list-item-action wux-list-item-focus">Focus</li>
</ul>

Special Style

Use wux-list-item-action to apply the mouse hover style, and also use wux-list-item-focus to apply the mouse focus style after application.
