WuX-UI Form-Checks

The Form-Checks component is used when multiple options can be selected in the selection.

Example

<label>
  <input class="wux-form-checks" type="checkbox" />
  Test!
</label>

Customize

SCSS Variables

/*Default*/
$form-checks-border-color
$form-checks-border-radius

$form-checks-hover-background-color
$form-checks-hover-shadow

$form-checks-checked-background-color
$form-checks-checked-border-width
$form-checks-checked-border-color
$form-checks-checked-shadow

/*Dark*/
$form-checks-border-color-dark

$form-checks-hover-background-color-dark
$form-checks-hover-shadow-dark

$form-checks-checked-background-color-dark
$form-checks-checked-border-color-dark
$form-checks-checked-shadow-dark
