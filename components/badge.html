WuX-UI Badge

The Badge component is a simple and clear numeric prompt in the top right corner of the component.

Example

<button class="wux-btn">
  Button with Badge
  <span class="wux-badge">12</span>
</button>

Customize

SCSS Variables

/*Default*/
$badge-font-size
$badge-padding
$badge-background-color
$badge-text-color

/*Dark*/
$badge-background-color-dark
$badge-text-color-dark

CSS Variables

:root {
  --badge-font-size
  --badge-padding
  --badge-background-color
  --badge-text-color
}
