Also need Polyfill...

Although HTML 5 native Dialog is currently supported by the latest browsers, Firefox and Safari are not yet supported in older versions (e.g. Safari 15.3 and Firefox 97) (even though Chrome has supported it for several years), see MDN for details.

We suggest using a11y-dialog in MDN, but since this implementation is much different from the original, we suggest using GoogleChrome/dialog-polyfill directly, which is well supported in WuX-UI.

Therefore, for developers' convenience, dialog-polyfill and its LICENSE are included in /dist/polyfill/dialog-polyfill.
