WuX-UI Header

The Header component is placed at the top of the page and is used to navigate the page.

Example

Title
  • Option
  • Bold
<header class="wux-header">
  <span class="wux-header-title">Title</span>
  <button class="wux-header-small-option-group">Menu</button>
  <ul class="wux-header-option-group">
    <li class="wux-header-option" href="https://github.com/wux-ui/wux-ui">Option</li>
    <li class="wux-header-option wux-header-option-bold" href="https://wux-ui.tk/">Bold</li>
  </ul>
</header>

Fix it

The component has a wux-header-fixed class to hold it at the top, but the Body must use the wux-header-fixed-margin class.
