WuX-UI Blankslate

The Blankslate component provides error messages when the original information is missing.

Example

This is a Blankslate!

Yeah, Blankslate!

<div class="wux-blankslate">
  <img src="https://avatars.githubusercontent.com/u/108740219?s=200&v=4" class="wux-blankslate-icon"/>
  <h1 class="wux-blankslate-title">This is a Blankslate!</h1>
  <p class="wux-blankslate-subtitle">Yeah, Blankslate!</p>
  <div class="wux-blankslate-button-group">
    <button class="wux-btn">Great!</button>
    <button class="wux-btn wux-btn-text">Uhh...</button>
  </div>
</div>

Customize

SCSS Variables

/*Default*/
$blankslate-width

$blankslate-title-font-size
$blankslate-title-text-color

$blankslate-subtitle-font-size
$blankslate-subtitle-text-color

$blankslate-icon-size
$blankslate-icon-border-radius

/*Dark*/
$blankslate-title-text-color-dark
$blankslate-subtitle-text-color-dark

CSS Variables

:root {
  --blankslate-width

  --blankslate-title-font-size
  --blankslate-title-text-color

  --blankslate-subtitle-font-size
  --blankslate-subtitle-text-color	

  --blankslate-icon-size
  --blankslate-icon-border-radius
}
