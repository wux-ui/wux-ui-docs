WuX-UI Button

The Button component is used to guide the user through the most basic actions.

Example

<button class="wux-btn">
  Click Me~
</button>

Colors

The default Button color is Primary, and you can change the Button color using wux-btn-[color], see Customize/Colors for details.

Size

Button has five sizes (xs, sm, md, lg, xl), use wux-btn-[size] to apply the style.

The default size is Medium.

Styles

Button has three basic styles, which are wux-btn-solid, wux-btn-outline, wux-btn-text.

Border-Styles

Button has three border-styles, which are wux-btn-square, wux-btn-default, wux-btn-round.

Customize

$button-outline-border-width
$button-margin-right
$button-margin-bottom

$button-fab-shadow

// Color-Degree
$button-background-degree
$button-border-degree

/*Dark*/

$button-fab-hover-shadow-dark

// Color-Degree
$button-background-degree-dark
$button-border-degree-dark
