WuX-UI Form-Input

The Form-Input component is an input box, which can be divided into single line and multi-line.

Example

<input class="wux-form-input" placeholder="md"/>

Size

Button has five sizes (xs, sm, md, lg, xl), use wux-form-input-[size] to apply the style.

The default size is Medium.

Disabled

The Form-Input component supports the use of HTML's Disabled attribute.

Validation

The Form-Input component supports type validation using HTML's built-in type validation mechanism.

Textarea

The Form-Input component can also be used in a textarea.
