WuX-UI Components

Edit Style

Edit the files in /src/customize/components.

The customization structure of a component is

  • Default SCSS Variables
  • Dark Mode SCSS Variables
  • Default CSS Variables
  • [Dark-Mode] CSS Variables
  • @Media(Refers-Color-Scheme) CSS Variables

However, some components do not have the dark mode variables.

You can also refer to the definition of each variable at the bottom of each component documentation.
