WuX-UI Color

About Default Color

WuX-UI has 7 built-in status colors, they are:

Primary

which indicates the primary operation.

Secondary

Indicating secondary operation.

Success

Indicating successful or correct operation, etc.

Warning

Which indicates a warning, dangerous operation.

Error

Indicating that there is an error.

Info

Used to prompt, or see more information.

Light

Can replace Secondary.

Degrees

WuX-UI has nine selectable saturation levels for each color.

Use

In some components (Alert, Button, Tag) you can use Class directly to apply the color.

You can also apply colors using CSS variables in components that do not support applying colors using Class.

Edit Colors

Edit the files in /src/customize/color/colors.

The color map will be generated automatically when you modify $color-(color name), or you can modify the map yourself.

Dark Mode

You can use the class wux-dark-mode or dark-mode property in the body.

Most WuX-UI components currently have dark mode support, and we plan to add dark mode detection to Utilities for a better experience.
